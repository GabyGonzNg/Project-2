# Project-2
The Nebraska Department of Transportation has contracted our firm to modernize the concrete mix design process. The current system relies on an Excel-based workflow, which NDOT wants to migrate to a Python-based system. The objective is to replicate and automate the logic found in the “Mix Design” worksheet found in the existing notebook. This transition is intended to improve reliability by moving calculations from an offline spreadsheet to a Python program ensuring that the logic is fixed and repeatable.
